# Privacy Policy

Effective Date: [Insert Date]

Magnyte Software Pvt. Ltd. ("we", "our", "us") operates the MySecureFolder desktop application.

This Privacy Policy explains how information is collected, used, and protected when you use MySecureFolder.

---

## Information We Collect

### Account Information
- Email address
- Account identifier
- Subscription status

### Application Data
- Application configuration settings
- License information
- Audit logs (stored locally by default)

### Cloud Features (Optional)
If cloud features are enabled:
- Encrypted backup metadata
- Device identifiers for synchronization

We **do not** access or store unencrypted user files.

---

## How We Use Information

Collected information is used to:
- Authenticate users
- Manage subscriptions and licenses
- Provide cloud backup and restore features
- Improve application reliability and security

---

## Data Storage & Security

- Encryption keys are generated and handled locally
- Encrypted data remains encrypted during transmission and storage
- Core folder protection works offline
- Cloud features are optional and user-initiated

---

## Data Sharing

We do **not** sell or share personal data with third parties, except:
- When required by law
- To provide essential service infrastructure (hosting, email delivery)

---

## User Responsibilities

Users are responsible for:
- Securing their master PIN or password
- Maintaining access to their registered email
- Enabling backups if data recovery is required

Loss of authentication credentials may result in permanent data loss.

---

## Changes to This Policy

This Privacy Policy may be updated periodically. Changes will be published on our official website.
