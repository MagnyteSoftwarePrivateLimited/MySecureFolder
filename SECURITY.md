# Security Policy

## Supported Versions

The following versions of MySecureFolder are currently supported with security updates:

| Version | Supported |
|--------|-----------|
| 1.x.x  | ✅ Yes    |
| < 1.0  | ❌ No     |

---

## Reporting a Vulnerability

If you discover a security vulnerability in MySecureFolder, please report it responsibly.

### How to Report
- Email: security@magnytesolution.com
- Subject: **Security Vulnerability Report – MySecureFolder**

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Potential impact
- Screenshots or logs (if available)

---

## Disclosure Policy

- We will acknowledge your report within **72 hours**
- We will investigate and validate the issue
- Fixes will be released in a future update
- Public disclosure should only occur after a fix is released

---

## Security Scope

This policy applies to:
- MySecureFolder desktop application
- Official releases published by Magnyte Software Pvt. Ltd.
- Backend services directly operated by Magnyte Software Pvt. Ltd.

Third-party dependencies and external services are subject to their respective security policies.

---

Thank you for helping keep MySecureFolder secure.
